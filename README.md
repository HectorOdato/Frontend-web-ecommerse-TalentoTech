# Tienda Online - Proyecto Pre entrega TalentoTech
Esta tienda e-commerse esta desarrollada con HTML,CSS y JavaScript permite visualizar productos,encabezado,listas,formulario y informacion de contacto,apartado de reseñas

Instrucciones o capturas de pantalla mostrando cómo usar la app.
## Uso
1. Ingresá al sitio.
2. Navegá por los productos.
3. agrega productos al carrito de compras
4. Completa formulario de contacto

## Tecnologías
- HTML
- CSS (Flexbox, Grid, Media Queries)
- JavaScript

## Características
- Logo y titulo de la web
- Lista de productos
- Validación de formularios
- Seccion de reseñas
- Informacion de contacto
- Diseño responsive
- Productos consumidos por una API
- carrito de compras
